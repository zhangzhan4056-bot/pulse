# 反脆弱策略深度学习指南

> 基于纳西姆·塔勒布 (Nassim Taleb) 的反脆弱理论，构建从混乱中受益的投资组合。

---

## 一、什么是反脆弱？

### 1.1 三态模型

塔勒布把事物分为三种状态：

| 状态 | 受到冲击时 | 例子 |
|------|-----------|------|
| **脆弱** | 受损 | 瓷杯、杠杆交易、集中持仓 |
| **强韧** | 不变 | 石头、分散化投资 |
| **反脆弱** | 变强 | 肌肉、免疫系统、期权买方 |

**核心洞察**：大多数投资组合是脆弱的——平时稳定，遇到黑天鹅时崩溃。反脆弱的目标是反过来——平时小亏，遇到黑天鹅时大赚。

### 1.2 反脆弱的数学本质

反脆弱 = **凸性收益结构**（Convexity）

```
收益
  |        /
  |       /
  |      /
  |     /
  |    /
  |   /
  |--/--------→ 损失
  | /
  |/
```

- **线性**：赚 10% 和亏 10% 对称
- **凹性**（脆弱）：赚 10% 的概率低，亏 10% 的概率高（大多数策略）
- **凸性**（反脆弱）：亏的时候亏有限，赚的时候赚无限

**关键公式**：
```
凸性 = 期权买方的收益结构
- 最大损失 = 权利金（已知、固定、有限）
- 潜在收益 = 无限（标的跌得越多，看跌期权赚得越多）
```

## 二、杠铃策略

### 2.1 核心思想

塔勒布的杠铃策略：**极端保守 + 极端激进，不要中间态**。

```
资金分配：
|<--- 85-90% --->|<--- 10-15% --->|
   极度安全          极度冒险
   (国债/现金)       (深度虚值期权)
```

**为什么不选中间态？**
- 中等风险资产（如公司债、REITs）在危机中也会崩
- 中等收益不足以补偿尾部风险
- 杠铃结构确保：最坏情况 = 损失 10-15% 权利金，不会更多

### 2.2 与传统投资组合的对比

| | 传统 60/40 | 杠铃策略 |
|---|---|---|
| 牛市 | 赚钱 | 赚钱（主仓）- 小亏权利金 |
| 震荡 | 小赚 | 小亏权利金 |
| 暴跌 | 大亏（股票跌、债券也跌） | 大赚（期权非线性收益） |
| 最大损失 | 理论上无限 | 已知（权利金总额） |

## 三、期权的凸性

### 3.1 看跌期权的收益结构

买入 SPY 看跌期权（Put）：

```
收益
  |
  |  \
  |   \
  |    \
  |     \
  |      \___________
  |                  \_____
  |________________________\______→ SPY 价格
        行权价

- SPY 涨：最大亏损 = 权利金（比如 0.5%/月）
- SPY 小跌（<5%）：亏损 = 权利金 - 小额赔付
- SPY 大跌（>5%）：收益指数级放大
```

### 3.2 为什么是"深度虚值"？

深度虚值 (Deep OTM) = 行权价远低于当前价格

**例子**：SPY 当前 500，买行权价 475 的看跌期权（5% OTM）

- **便宜**：权利金低（~0.3%/月，因为行权概率低）
- **杠杆高**：如果 SPY 跌到 450，期权价值可能翻 10 倍以上
- **平时稳亏**：大多数月份 SPY 不会跌 5%，权利金全损
- **偶尔暴赚**：一旦 SPY 月跌 >5%，BS 模型给出的收益远超权利金（见第四章赔付表）

### 3.3 数学：小亏多次，大赚少次

当前参数：
- 月权利金成本：0.3%（每月支付，年化 ~3.6%）
- 对冲触发条件：SPY 月末收盘跌幅 > 5%
- 赔付公式：简化 Black-Scholes（见第四章）

**SPY 历史月度跌幅分布**（2018-2026）：

| 月份类型 | 概率 | 月收益 | 年化影响 |
|----------|------|--------|----------|
| SPY 涨或小跌 (<5%) | ~85% | -0.3%（权利金全损） | -3.1% |
| SPY 跌 5-10% | ~10% | +2%~+5% | +0.5%~+1% |
| SPY 跌 10%+ | ~5% | +7%~+25% | +0.5%~+2% |

**净效果**：正常年份年化拖累 ~3%，但大跌年份（2020、2022）对冲贡献远超成本。

### 3.4 真实历史回测验证

**2020 年（COVID 崩盘）**：
- 3 月 SPY 月末跌幅 -13%，BS 模型赔付 ~7.7%
- 全年对冲贡献：**+11.3%**（扣除全年成本 3.6%）

**2022 年（加息熊市）**：
- 多个月跌幅 5-10%，累计赔付 ~17%
- 全年对冲贡献：**+11.2%**（扣除全年成本 3.6%）

**正常年份（2021、2023、2024）**：
- 小跌月份赔付不足以覆盖成本
- 年化拖累 -2~4%

**累计效果（2019-2026）**：对冲净贡献约 +15%，相当于每年多赚 ~2%。

**关键结论**：反脆弱策略在**急跌**和**持续熊市**中都能提供保护，但正常年份有 ~3% 的保险成本。

## 四、Market Pulse 中的实现

### 4.1 策略结构

```python
class AntifragileStrategy:
    # 主仓：双动量策略（负责日常收益）
    # 尾部对冲：SPY 5% OTM 月 put（BS 模型模拟）

    hedge_monthly_cost = 0.003  # 每月 0.3% 权利金（年化 ~3.6%）
    hedge_otm_threshold = 0.05  # 5% OTM
    hedge_leverage = 15.0       # 向后兼容参数，BS 公式不依赖此值
```

### 4.2 回测引擎的对冲逻辑

```python
# 每个再平衡周期（月度）：

# 1. 扣除权利金
portfolio_value *= (1 - hedge_monthly_cost)  # -0.3%

# 2. 计算 SPY 月末收盘价跌幅
spy_monthly_return = (本月收盘 / 上月收盘) - 1

# 3. 判断是否触发对冲（跌幅 > 5% OTM 阈值）
if spy_monthly_return < -0.05:
    max_dd = abs(spy_monthly_return)
    payoff = _calc_option_payoff(max_dd, 0.05, 0.003)  # BS 公式
    portfolio_value *= (1 + payoff)
```

### 4.3 赔付公式：简化 Black-Scholes

核心思路：用 BS 公式计算 5% OTM 月 put 在月末结算时的理论价值。

```python
def _calc_option_payoff(monthly_drop, otm_threshold=0.05, cost=0.003, vol=0.20):
    """月末结算的 5% OTM 月 put 赔付。

    参数:
        monthly_drop: SPY 月末跌幅（正值，如 0.13 = 13%）
        otm_threshold: OTM 虚值比例（5%）
        cost: 月权利金占组合比例（0.3%）
        vol: 年化波动率（默认 20%，SPY 长期均值）

    流程:
        1. 设 S0=100, K=S0×0.95（行权价）
        2. 月末价格 S1 = S0×(1 - monthly_drop)
        3. BS 公式计算 put 理论价值
        4. 与内在价值取较大值
        5. 赔付 = (put价值 - 权利金) / S0
    """
    S0 = 100.0
    K = S0 * (1 - otm_threshold)
    S1 = S0 * (1 - monthly_drop)
    T = 1/12  # 1 个月

    d1 = (ln(S1/K) + 0.5×vol²×T) / (vol×√T)
    d2 = d1 - vol×√T

    put_value = K×e^(-rT)×N(-d2) - S1×N(-d1)
    put_value = max(put_value, max(K-S1, 0))  # ≥ 内在价值

    premium = cost × S0
    return (put_value - premium) / S0
```

**赔付表**（cost=0.3%/月）：

| SPY 月末跌幅 | 期权回报率 | 组合贡献 |
|:---:|:---:|:---:|
| 5% | 归零 | -0.3%（成本） |
| 6% | 757% | +2.3% |
| 8% | 1170% | +3.5% |
| 10% | 1667% | +5.0% |
| 13% | 2567% | +7.7% |
| 20% | 4900% | +14.7% |
| 30% | 8233% | +24.7% |

### 4.4 参数说明

| 参数 | 值 | 含义 |
|------|-----|------|
| `hedge_monthly_cost` | 0.3%/月 | 每月支付的权利金（年化 ~3.6%） |
| `hedge_otm_threshold` | 5% | SPY 月末跌幅超过此值才触发对冲 |
| `vol` | 20% | BS 公式用的年化波动率（SPY 长期均值） |

**参数敏感性**：
- 权利金从 0.3% 提到 0.5% → 年化拖累从 3.6% 增到 6%，正常年份更难回本
- OTM 阈值从 5% 降到 3% → 触发更频繁，但赔付更小，且权利金成本可能上升
- 波动率从 20% 降到 15% → BS 定价更低，赔付减少（实际中低波动率=便宜期权）

### 4.5 当前实现的局限性

**模拟 vs 真实期权**：
- BS 公式假设固定波动率（20%），真实 IV 随市场情绪变化
- 高 IV 时期权更贵（如 VIX > 30 时），固定 0.3% 可能低估成本
- 未考虑 Theta 衰减（期权到期前时间价值加速流失）
- 未考虑买卖价差和流动性成本

**改进方向**：
1. 接入 VIX 数据，根据 IV 水平动态调整权利金成本和 BS 波动率参数
2. 使用真实期权链数据回测（ORATS $99/月、CBOE DataShop）
3. 优化行权价选择（固定 OTM % vs 固定 delta）
4. 加入期权到期日管理（月度 vs 季度）

## 五、与其他策略的关系

### 5.1 反脆弱 vs 双动量

反脆弱策略的**主仓就是双动量**。区别：

| | 双动量 | 反脆弱 |
|---|---|---|
| 熊市应对 | 切现金（绝对动量过滤） | 切现金 + 期权对冲收益 |
| 暴跌时 | 避开（如果信号及时） | **主动获利**（期权赔付） |
| 成本 | 无额外成本 | 每月 0.3% 权利金（年化 3.6%） |
| 适合场景 | 趋势性熊市 | 急跌 + 持续熊市 |

**实际回测对比**（2019-2026）：

| 指标 | 双动量 | 反脆弱 | 差异 |
|------|:---:|:---:|:---:|
| 总收益 | 230% | 261% | **+31%** |
| 年化收益 | 20.7% | 22.4% | **+1.7%** |
| 夏普 | 0.93 | 1.01 | **+0.08** |
| 最大回撤 | -20.8% | -21.1% | -0.3% |

对冲在 2020（+11.3%）和 2022（+11.2%）贡献显著，正常年份拖累 -2~4%。

### 5.2 反脆弱 vs 回撤控制

两者都是「保本优先」策略，但方法不同：

- **回撤控制**：被动防御——回撤超阈值就减仓
- **反脆弱**：主动进攻——暴跌时期权赚钱

### 5.3 最佳组合

反脆弱策略不应单独使用，而是作为组合中的「保险层」：

```
推荐组合：
- 核心仓位（80%）：双动量 / 回撤约束优化
- 保险层（20%）：反脆弱策略的对冲部分
```

这样既有日常收益，又有黑天鹅保护。

## 七、关键概念速查

| 概念 | 解释 |
|------|------|
| **凸性** | 收益曲线向上弯曲，损失有限、收益无限 |
| **凹性** | 收益曲线向下弯曲，收益有限、损失无限 |
| **杠铃** | 极端保守 + 极端激进，避免中间态 |
| **深度虚值 (Deep OTM)** | 行权价远低于当前价格的期权 |
| **权利金** | 买期权支付的费用（最大损失） |
| **Theta** | 期权时间价值的衰减率（对买方不利） |
| **Vega** | 期权价格对波动率的敏感度 |
| **IV (隐含波动率)** | 市场对未来波动率的预期 |
| **Gamma** | Delta 的变化率（凸性的来源） |
| **尾部风险** | 极端事件导致的非正态分布损失 |
